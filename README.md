# 🛍️ Customer Segmentation Using K-Means Clustering

## 🚀 SkillCraft Technology - Task 02

Successfully completed **Task 02** as part of my **Machine Learning Internship at SkillCraft Technology**.

## 📌 Project Overview

This project implements **K-Means Clustering** to segment mall customers into different groups based on their **Annual Income** and **Spending Score**.

Customer segmentation helps businesses understand customer behavior, identify target audiences, and create personalized marketing strategies.

## 📂 Project Files

### 📄 mall_customer_segmentation.py

Core Machine Learning implementation that:

- Loads and preprocesses the dataset
- Applies K-Means Clustering
- Uses the Elbow Method to determine optimal clusters
- Visualizes customer segments
- Displays cluster centroids

### 🌐 app.py

Interactive Streamlit dashboard that provides:

- Sidebar controls for cluster selection
- Interactive Plotly visualizations
- Customer insights and analytics
- Income and spending distributions
- Real-time cluster exploration

## 📊 Dataset

**Mall Customer Segmentation Dataset**

Dataset Features:

- CustomerID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1-100)

Dataset Source:
https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python
